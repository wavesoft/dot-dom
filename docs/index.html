Introduction

This section demonstrates some introductory usage of .dom

Hello World

The simplest way you can test the library is to create a div element with a simple text content. The function H() can be used to create elements and the function R() can be used to render them in the DOM.

          R(
            H("div", "Hello world"),
            document.body
          );

Stateless Components

You can create simple stateless components as functions that return the render contents. The first argument is the properties that were given at construction time.

          function Hello(props) {
            return H('div', `Hello ${props.to}`);
          }

          R(
            H(Hello, { to: 'There' }),
            document.body
          );

Stateful Components

It is also quite simple to create stateful components. The component state is given as a second argument and the setState function is given as a third. Calling the setState function will patch the state and re-render your component with the new state.

          function Clickable(props, state, setState) {
            const {clicks=0} = state;

            return H('button',
              {
                onclick() {
                  setState({clicks: clicks+1})
                }
              },
              `Clicked ${clicks} times`
            );
          }

          R(
            H('div',
              H(Clickable),
              H(Clickable)
            ),
            document.body
          )
